ST_analysisは、測定した4chのwavファイルを入力データに用いて、独自に構築した指標を算出するプログラムです。

測定位置一つにつき、前後左右上下の六方向の値が、解析対象のオクターブバンドの数だけ出力されるので、測定位置ごとにMeasurementPointクラスのインスタンスを作って値（と途中の信号処理の経過）を管理しています。
また、測定会場ごとに複数の測定位置を持つので、MeasurementPointクラスのインスタンスはVenueクラスのインスタンスで管理しています。

AtoB関数、Bto6関数は、ambisonicsの理論に基づいて実測データから方向別の測定データに変換する関数です。
mk_BPF関数は、周波数分析をするために用いるバンドパスフィルタを作成する関数です。
riseFind関数は、インパルス応答の直接音の到来時刻（サンプル番号）を検出する関数です。
